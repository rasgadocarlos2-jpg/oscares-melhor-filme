# Sistema de Gestão dos Óscares

Trabalho de universidade em Python para gerir filmes nomeados aos Óscares (categoria Melhor Filme).

O repositório contém:
- Relatório em PDF com análise, fluxogramas e testes.
- Código (em anexo no relatório ou noutra versão do projeto).

Objetivo: praticar conceitos de algoritmos, estruturas de dados, ordenação e persistência em ficheiros CSV.
